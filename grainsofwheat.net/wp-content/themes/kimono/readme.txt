== Instruction ==
How to set slideshow on homepage:
Creat category named "news" to make a post displayed as top slider. 
?
How to set logo:
Please choose from Media Library, not from the file stored in your computer
?
How to set menu:
Menu setting in Appearance area is not supported.Please use custom menu function in widget area.
?
Appearance>edit menus
Create new menu
Appearace>Widgets
select the menu you created in custom menu 
?
